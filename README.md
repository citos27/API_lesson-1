# Прогноз погоды

Скрипт выводит прогноз погоды на 3 дня в терминал, используя сайт [wttr.in](https://wttr.in)

### Как установить

На компьютере должен быть устнаовлен Python3, а так же потребуется установить библиотеку Requests

Для установки Requests просто выполните эту простую команду в любом терминале:
```
python -m pip install requests
```

### Пример запуска скрипта:

Для запуска скрипта введите в терминале:
```
python3 main.py
```
